  
В рамках исследования были проанализированы 10 наиболее популярных систем, используемых для работы с документацией и диаграммами, по следующим критериям:
- поддержка текстового и графического редактирования;
- совместная работа в реальном времени;
- расширяемость (добавление новых типов элементов);
- интеграция с PlantUML/Structurizr;
- модель лицензирования.

**Таблица 1.1 – Анализ аналогов**

| №   | Аналог               | Ключевые особенности                            | Плюсы                            | Минусы                                        |
| --- | -------------------- | ----------------------------------------------- | -------------------------------- | --------------------------------------------- |
| 1   | **Miro**             | Бесконечное полотно, совместная работа, шаблоны | Удобство, производительность     | Слабая документация, нет текстового редактора |
| 2   | **Confluence**       | Wiki-документация, пространства, макросы        | Интеграция с Jira, структура     | Дорогой, сложный, нет диаграмм                |
| 3   | **Google Docs**      | Совместный текстовый редактор                   | Скорость, комментарии, история   | Нет диаграмм, примитивные таблицы             |
| 4   | **Draw.io**          | Диаграммы, интеграция с Confluence              | Много нотаций, экспорт           | Нет совместной работы                         |
| 5   | **Mermaid/PlantUML** | Диаграммы из кода                               | Git-friendly, текстовое описание | Неудобно для сложных схем                     |
| 6   | **Notion**           | База знаний, базы данных, блоки                 | Гибкость, UX                     | Нет диаграмм, платная версия                  |
| 7   | **Bizagi Modeler**   | BPMN моделирование                              | Профессиональный BPMN            | Тяжелый, Windows-only                         |
| 8   | **Figma**            | Дизайн, прототипирование                        | Компоненты, плагины              | Не для документов                             |
| 9   | **Structurizr**      | C4 моделирование                                | Текст + диаграммы                | Специфичен для C4                             |
| 10  | **Obsidian**         | Markdown, граф знаний                           | Быстрый, плагины                 | Нет совместной работы                         |
