В приложении должны быть размещены скриншоты основных экранов разработанной клиентской части. Иллюстрации подтверждают реализацию пользовательских сценариев и позволяют визуально сопоставить проектные решения с итоговым интерфейсом.

Рекомендуемый состав иллюстраций:

| Номер рисунка | Экран | Назначение |
| --- | --- | --- |
| Г.1 | Главная или стартовая страница приложения | Демонстрация точки входа в систему |
| Г.2 | Список последних изменений | Отображение навигации и проектного контекста |
| Г.3 | Страница проекта | Демонстрация файловой структуры и действий над документами |
| Г.4 | Блочный текстовый редактор | Подтверждение реализации текстового редактирования |
| Г.5 | Графический редактор диаграмм | Подтверждение реализации холста, узлов и связей |
| Г.6 | Инспектор свойств элемента диаграммы | Демонстрация редактирования параметров нотации |
| Г.7 | Переключение языка интерфейса | Подтверждение работы локализации |
| Г.8 | Темная тема интерфейса | Демонстрация альтернативного цветового режима |

После финальной сборки проекта скриншоты следует сформировать в одинаковом разрешении, чтобы приложение выглядело целостно. Для каждого изображения в основном тексте диплома необходимо указать ссылку вида «рисунок Г.1».
