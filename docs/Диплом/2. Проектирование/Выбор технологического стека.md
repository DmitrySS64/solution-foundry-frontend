Выбран React 18 по следующим причинам:
1. **Наличие зрелых решений для редакторов** – Lexical (текстовый), React Konva (графический) и Yjs (совместная работа) имеют официальную или хорошо поддерживаемую интеграцию с React. Для Vue и Angular аналогичные решения либо отсутствуют, либо требуют самостоятельных обёрток.
2. **Огромная экосистема** – React располагает крупнейшим сообществом, что упрощает поиск решений типовых задач (роутинг, управление состоянием, тестирование, стилизация).
3. **Поддержка TypeScript** – React имеет отличную поддержку типизации, критичную для сложных моделей данных (узлы диаграммы, связи, нотации).
4. **Порог входа** – средний, но за счёт использования в других проектах обучающийся уже обладает необходимыми компетенциями.
5. **Производительность** – достаточная для реализации заявленных сценариев (НФТ-01, НФТ-02) при правильной оптимизации (мемоизация, ленивая загрузка).

**Язык программирования:** **TypeScript 5.x** – обеспечивает статическую типизацию, снижая количество ошибок на этапе разработки, и поддерживает новейшие возможности ECMAScript.
