Инициализация проекта выполнялась с использованием сборщика Vite. Выбор Vite обусловлен высокой скоростью холодного старта и мгновенной горячей заменой модулей (HMR), что критически важно для разработки редакторов с интенсивным взаимодействием.

  
Технологический стек окружения:

| Инструмент   | Версия | Назначение              |
| ------------ | ------ | ----------------------- |
| Node.js      | 20.x   | Среда выполнения        |
| npm          | 10.x   | Менеджер пакетов        |
| Vite         | 5.x    | Сборщик                 |
| TypeScript   | 5.x    | Типизация               |
| ESLint       | 8.x    | Статический анализ кода |
| Prettier     | 3.x    | Форматирование кода     |
| Husky        | 8.x    | Git-хуки (pre-commit)   |

**Команда создания проекта:**

```bash
npm create vite@latest solution-foundry-frontend -- --template react-ts
cd solution-foundry-frontend
npm install
```

В vite config настраиваем плагины и сокращённые пути директорий под FSD архитектуру.
В tsconfig.app.json также настраиваем сокращенные импорты и линт
