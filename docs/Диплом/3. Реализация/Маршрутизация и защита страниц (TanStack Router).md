Маршрутизация реализована с использованием библиотеки TanStack Router, обеспечивающей полную типизацию параметров маршрутов и встроенную поддержку ленивой загрузки.

==Структура маршрутов==
```
src/app/routes/
├── __root.tsx          # Корневой маршрут (layout)
├── index.tsx           # Главная страница
├── auth/
│   ├── login.tsx       # Вход
│   └── register.tsx    # Регистрация
├── projects.tsx        # Список проектов
├── project/
│   └── $id.tsx         # Страница проекта (динамический параметр)
└── document/
    └── $id.edit.tsx    # Редактирование документа
```
