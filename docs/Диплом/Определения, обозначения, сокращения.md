API (Application Programming Interface) - программный интерфейс, с помощью которого одна часть системы обращается к другой и получает доступ к ее функциям или данным.

CRDT (Conflict-free Replicated Data Type) - тип структуры данных, позволяющий нескольким пользователям изменять один документ независимо, а затем автоматически объединять изменения без ручного разрешения конфликтов.

Feature-Sliced Design (FSD) - архитектурная методология организации фронтенд-проекта, при которой код разделяется по слоям и функциональным областям приложения.

SPA (Single Page Application) - одностраничное веб-приложение, загружающее одну HTML-страницу и обновляющее интерфейс без полной перезагрузки браузера.

UI (User Interface) - пользовательский интерфейс, включающий визуальные элементы и способы взаимодействия пользователя с приложением.

URL (Uniform Resource Locator) - адрес ресурса в сети или внутри веб-приложения.

Бандл - итоговый набор файлов, сформированный сборщиком из исходного кода приложения и подключаемых зависимостей.

Веб-приложение - прикладная программа, доступная пользователю через браузер и выполняющая часть логики на клиентской стороне.

Графическая нотация - система условных обозначений, применяемая для построения диаграмм определенного типа, например BPMN, UML или C4.

Компонент - самостоятельная часть пользовательского интерфейса, которая описывает внешний вид и поведение отдельного элемента или группы элементов.

Локализация - адаптация интерфейса приложения к языку и культурным особенностям пользователя.

Маршрутизация - механизм сопоставления адресов приложения с отображаемыми страницами и компонентами.

Репозиторий - хранилище исходного кода проекта, включающее историю изменений и структуру файлов.

Тонкий клиент - клиентская часть системы, основная задача которой состоит в отображении интерфейса и передаче запросов к другим уровням приложения.

Фреймворк - программная платформа, задающая общую структуру приложения и предоставляющая набор готовых механизмов для разработки.

Хранилище состояния - часть приложения, предназначенная для централизованного хранения данных интерфейса и управления их изменением.

Хук - функция React, позволяющая использовать состояние, жизненный цикл компонента и другие возможности библиотеки внутри функциональных компонентов.
